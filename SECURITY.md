## Reporting a Vulnerability

Please do not open a public GitHub issue for security problems.

Instead, report vulnerabilities privately via:
- GitHub's Private Vulnerability Reporting (Security tab → Report a vulnerability), or
- Email: security@yourproject.org

Include:
- A description of the issue and its impact
- Steps to reproduce
- Affected version(s)
- Any suggested mitigation

## Response Process
- We will acknowledge your report within 3 business days.
- We aim to provide a status update within 7 days.
- Once a fix is ready, we'll coordinate disclosure with you via a GitHub Security Advisory.
